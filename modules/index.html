Modules

The PiWeb plot extension API has a root module named piweb which encapsulates all child modules. The following child modules are available:

Module Description
data Access the databinding of the plot element
drawing Draw the content of the plot extension element
environment Access settings of the PiWeb host application
events React to events emitted by the PiWeb host application
expressions Evaluate system variable expressions
format Read and write localized data
logger Write entries into the piweb log
properties Access the plot extension elements property grid
resources Access resources that are located in the extension package
tooltips Create tooltip content for the PiWeb Monitor info mode
